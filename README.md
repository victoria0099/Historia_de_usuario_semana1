# Product Cost Calculator
## Description

This project is a simple Python program created for a development course.
It is a basic calculator that helps the user compute the total cost of a product.

The program asks the user for the product name, the unit price, and the quantity. It then calculates the total cost based on the information provided.

## How it works

**1.** The program starts running in the Python environment.

**2.** The system asks the user to enter the product name.

**3.** The program asks the user to enter the unit price of the product.

**4.** The program asks the user to enter the quantity of the product.

**5.** The system converts the unit price to a float value.

**6.** The program converts the quantity to an integer.

**7.** The system multiplies the unit price by the quantity.

**8.** The program calculates the total cost.

Finally, the system displays the product information and the total cost to the user.

The program collects user input and processes the information step by step. It uses simple arithmetic operations to calculate the result.

## Status

The project is currently running as a basic prototype and is being used to practice Python programming and documentation skills.
